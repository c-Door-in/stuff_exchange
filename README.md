# Проект обмена вещами Stuff Exchange Bot
 

Проект позволяет обмениваться вещами посредством телеграм-бота.

## Как установить

Для запуска проекта необходимо выполнить последовательность действий:


### 1. Регистрируем нового Телеграм-бота

Для регистрации необходимо зайти в специального телеграм-бота [@BotFather](https://telegram.me/BotFather) и ввести следующую команду:

```
/newbot
```
BotFather предложит задать имя и username для нового бота. Задаем и получаем в ответном сообщении токен для API. Он выглядит примерно так: `958423683:AAEAtJ5Lde5YYfkjergber` (токен нерабочий и представлен для ознакомления).

### 2. Создаем в корне проекта файл .env

В файл нужно поместить полученныЙ при создании бота токен в следующем формате:

```
TG_TOKEN=
```


Python3 должен быть уже установлен. 
Затем используйте `pip` (или `pip3`, если есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```
## Как запустить

Для запуска бота необходимо выполнить следующую команду:

```
python3 bot.py
```

## Функционал бота

Для запуска необходимо ввести команду:
```
/start
```
После этого вы увидите приветственное сообщение и три кнопки внизу:

- Добавить вещь
- Найти вещь
- Обменяться

### 1. Добавить вещь

При нажатии на кнопку бот предложит загрузить фото с описание (поле "Добавить подпись" или "Caption"). После успешного добавления бот выведет сообщение `Добавлено`.

### 2. Найти вещь

Если вы еще не добавляли собственные вещи, то вы увидите сообщение `Сперва нужно добавить свою вещь`. После добавления вещи бот выведет случайное изображение вещи с подписью от другого пользователя.

### 3. Обменяться

Если вам понравилась вещь другого пользователя, нажмите эту кнопку, бот выведет сообщение `Успешно добавлено в список желаний`. Если ваша вещь также понравится владельцу этой вещи, бот выведет вам обоим сообщение с фотографиями этих вещей и контактами.
